# Blog-Framework
hexo+yilia主题博客框架

# 步骤
## 1、git clone
```
    git clone git@github.com:gdutewei/Blog-Framework.git
```
## 2、进入项目目录，安装依赖
```
    npm install
```
## 3、本地运行
```
    hexo clean  // 清除hexo缓存
    hexo g  // 生成静态文件
    hexo s // 启动本地服务
```
## 4、查看打开浏览器
```
    http://localhost:4000
```
## 5、上传到github的page页
```
    hexo d
```
就可以看到自己搭建的技术博客了 https://gdutewei.github.io/
